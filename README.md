Project Overview

About University â€“ UET Mardan is a static HTML webpage that provides an overview of the University of Engineering & Technology (UET), Mardan, including its background, academic departments, academic objectives, affiliations, and campus media.
The project is intended as an academic exercise to demonstrate understanding of basic HTML structure while presenting institutional information in an organized and readable format.

The webpage uses only HTML and integrates text, lists, images, and videos without the use of CSS or JavaScript.

Objectives

Practice and demonstrate fundamental HTML concepts

Present official and academic information about UET Mardan

Organize content using headings, lists, and sections

Include multimedia elements using standard HTML tags

Create a simple informational webpage suitable for academic submission
The webpage is divided into multiple logical sections:

1. About the University

Introduces the University of Engineering & Technology, Mardan

Mentions its location in Khyber Pakhtunkhwa, Pakistan

Highlights recognition by HEC and PEC

Includes a brief student introduction

2. Academic Strengths and Learning Outcomes

Emphasizes:

Program Learning Outcomes (PLOs)

Course Learning Outcomes (CLOs)

Practical work and laboratory-based learning

Highlights the importance of student effort and project-based learning

3. MOUs and Accreditation

Engineering programs registered with Pakistan Engineering Council (PEC)

Academic programs recognized by Higher Education Commission (HEC)

Internship MOUs signed with reputed organizations

4. Academic Departments

The webpage lists and briefly explains major departments, including:

Electrical Engineering

Telecommunication Engineering

Computer Software Engineering

Computer Science

Civil Engineering

Mechanical Engineering

Natural Sciences and Humanities

5. Special Centers

Center of Artificial Intelligence

Focus on AI, Machine Learning, and Data Science research and training

6. Programs Offered

Undergraduate programs (BS/BSc Engineering)

Postgraduate programs (MS/MSc)

Doctoral programs (PhD)

7. Campus Media

Images of the university campus and academic block

Videos showcasing the university environment and facilities

Technologies Used

HTML5

HTML elements such as:

Headings and paragraphs

Ordered and unordered lists

Sections and divs

Images (img)

Videos (video)

Horizontal rules (hr)

No external styling libraries, CSS, or JavaScript were used.

How to Run the Project

Download or clone the repository

Ensure the media folder exists as referenced in the HTML file:

Images and Videos/ (contains images and video files)

Open the About University.html file in any modern web browser
